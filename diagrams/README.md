# Diagrams — NEXUS-32 Specification

This directory holds **architecture diagrams** and **memory maps** for the NEXUS-32 fantasy game console, as described in the main specification §13.1 (Repository Layout).

Contents are intended to support:

- System bus and component interconnection (see spec §1.1–1.2)
- Memory map and region layout (see spec §3)
- Other visual aids referenced by [NEXUS32_Specification_v1.0.md](../NEXUS32_Specification_v1.0.md)

Add diagram files here as they are produced; keep naming consistent and reference the relevant spec sections.
